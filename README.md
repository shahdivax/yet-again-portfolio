# My AI-Powered Portfolio

A modern, interactive portfolio showcasing projects and skills with a strong focus on **Google Generative AI (Gemini)**. Built with the latest web technologies, this portfolio features dynamic UI, responsive design, and highlights key contributions in AI development.

## Highlights

- **Google Generative AI Integration:** Demos and projects leveraging Google Gemini for various AI capabilities.
- **Modern Tech Stack:** Built with Next.js, React, and TypeScript for a robust and scalable frontend.
- **Dynamic User Interface:** Smooth animations and transitions powered by Framer Motion.
- **Responsive Design:** Optimized for a seamless experience across all devices.

- **Clean and Maintainable Code:** Styled with Tailwind CSS for efficient and consistent design.

## Tech Stack

- **Frontend:** Next.js 16, React 19, TypeScript
- **Styling:** Tailwind CSS 4, PostCSS
- **Animations:** Framer Motion
- **AI Integration:** Google Generative AI (`@google/genai`)
- **Utilities:** `clsx`, `lucide-react`, `next-themes`, `react-markdown`, `remark-gfm`, `tailwind-merge`

## Getting Started

First, install the dependencies:

```bash
npm install
# or yarn install
# or pnpm install
# or bun install
```

Then, run the development server:

```bash
npm run dev
# or yarn dev
# or pnpm dev
# or bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
